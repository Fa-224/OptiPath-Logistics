Dataset Information
* Original Source: E-Commerce & Logistics Dataset on Kaggle
  https://www.kaggle.com/datasets/davidafolayan/e-commerce-dataset
* Original Data: Includes 100,000+ transaction records.
* Processed Data: The file cleaned_logistics_data.csv contains a sampled version of 10,000 records, cleaned from missing values (NaN) in coordinates, specifically prepared for this project's algorithms.
